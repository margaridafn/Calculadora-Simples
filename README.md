# Projeto 2.1: Calculadora Simples

`Objetivo:` Criar uma calculadora que realiza as quatro operações aritméticas básicas com base
em dois números fornecidos pelo utilizador.

`Requisitos:`
1. Use a função input() para pedir ao utilizador dois números.
2. Lembre-se que input() devolve strings. Converta os valores de entrada para um tipo
numérico (por exemplo, float()).
3. Armazene os números em variáveis.
4. Realize as operações de adição, subtração, multiplicação e divisão.
5. Use a função print() para exibir os resultados de forma clara.
